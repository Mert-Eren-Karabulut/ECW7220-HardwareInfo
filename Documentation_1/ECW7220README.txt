ECW7220 Wireless Access Point


The ECW7220 is a cost optimized Wi-Fi Access Point. The design specification as well as the entire design package has been contributed to Open Compute Project.   

The contributed files are:

BOM
Gerber file
PCB Schematic files
.brd design files
Mechanical STEP files
Mechanical assemble Drawings
Test Plan 

Edgecore Networks has made this Specification and Complete design package available under the Open Compute Project Hardware License (Permissive) Version 1.0, which is available at
       http://www.opencompute.org/community/get-involved/spec-submission-process/
